About the Library of Things

The Library of Things is a community-driven initiative that allows members to borrow tools, equipment, and other resources, promoting sustainability, saving costs, and fostering a sharing economy.

Our Mission

Our mission is to create an accessible platform where individuals can borrow items they need, reducing waste, and contributing to a more sustainable future.

Key Features

  • Browse a comprehensive catalog of items.
  • Filter by category, availability, and more.
  • Seamless reservation system for borrowing items.
  • User dashboard to manage reservations and history.

Project Insights

Task 1: Interview Insights

The interview with stakeholders revealed key requirements for the Library of Things, including an intuitive user interface, robust item catalog, and seamless reservation process. Stakeholders emphasized the importance of a community-driven approach to encourage collaboration and resource sharing.

Task 2: Requirements Specification

Based on the requirements, the system must include features such as category-based filtering, a user-friendly dashboard, and error handling mechanisms. Non-functional requirements highlighted the need for scalability, responsiveness, and accessibility to cater to diverse user needs.
